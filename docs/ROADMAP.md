# 🛣️ Roadmap — AprilDawn

Legend: ✅ done · 🟡 in progress / next · 🔭 later · 🔮 moonshot

## Phase 0 — Marketing MVP _(this repo)_ ✅
- ✅ Brand, design system, responsive marketing site
- ✅ 7 services + 3 flagship experience pages + shop + pricing
- ✅ Drag-and-drop uploader (client-side preview)
- ✅ Reminder + contact forms → API route stubs
- ✅ Amazon affiliate link builder + disclosure
- ✅ SEO (metadata, OG, sitemap, robots) + legal templates

## Phase 1 — Commerce & accounts (v1) 🟡
**Goal: take real orders and store real memories.**
- 🟡 Auth + accounts (email + social login)
- 🟡 **Memory Vault**: encrypted object storage, albums, sharing
- 🟡 Real uploads via pre-signed URLs; resumable/large-file + mobile capture
- 🟡 Catalog → cart → **Stripe checkout** + order management
- 🟡 **Family Vault** subscription billing
- 🟡 Print fulfillment integration (Printful/Printify) + live mockups
- 🟡 Product **designer/preview** with smart cropping
- 🟡 Transactional email/SMS (order updates, proofs)
- 🟡 Admin/ops console for the lab

## Phase 2 — The magic (v2) 🔭
**Goal: deliver the experiences only AprilDawn has.**
- 🔭 **AI restoration pipeline** (de-scratch, colorize, up-res) + proof approval UI
- 🔭 Audio/video **transcription & captioning**
- 🔭 **MemoryBox** mail-in kits: labels, intake scanning, item-level lab tracking
- 🔭 **Occasion integrations** (Google/Outlook/Apple/Facebook) + reminder scheduler + 1-tap gifting
- 🔭 **Living Wall** auto-collage layout engine + LED companion app + install ops
- 🔭 **Memory Mail** studio: NFC/QR/AR message embedding, scheduled sends, subscription card runs
- 🔭 **Hand-painted masterpieces** artist marketplace & commission workflow
- 🔭 **Family spaces**: shared vaults, collaborative walls, contributor invites

## Phase 3 — Scale & ecosystem 🔮
- 🔮 AR mobile app ("point at the photo, watch it move")
- 🔮 Living Portraits (consent-gated, labeled)
- 🔮 Smart gift concierge; one-click anniversary gag engine
- 🔮 Time-capsule scheduled sends; "Tell Me About This" elder interviews
- 🔮 B2B/Legacy: funeral homes, genealogists, museums, estates
- 🔮 Regional same-day maker network for cakes & rush gifts

## Guiding sequencing rule
Ship the **money path** (upload → vault → order → fulfill → deliver) before the **magic path** (Living Wall, AR, Living Portraits). The magic sells; the money path sustains.
